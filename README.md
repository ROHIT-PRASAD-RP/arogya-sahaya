# 🏥 Arogya Sahaya

Arogya Sahaya is an Android healthcare assistance application designed to help users manage medicines, health records, reminders, vitals tracking, and emergency support in a simple and user-friendly way.

---

# ✨ Features

## 🔐 Authentication

* Firebase Login & Registration
* Secure user authentication
* User profile management

## 💊 Medicine Reminder

* Add medicines with dosage and schedule
* Morning / Afternoon / Night reminders
* Mark medicines as Taken or Missed
* Reminder status tracking

## ❤️ Vitals Tracking

* Record:

  * Blood Pressure
  * Blood Sugar
  * Heart Rate
* Health monitoring dashboard

## 🚨 Emergency SOS

* One-tap SOS emergency feature
* Quick emergency assistance
* Emergency contact integration

## 👤 Profile Management

* User health profile
* Medical conditions
* Emergency contact information

## 🏥 Health Camp Information

* Upcoming camp notifications
* Health awareness support

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/ROHIT-PRASAD-RP/arogya-sahaya.git
```

---

## Open in Android Studio

1. Open Android Studio
2. Click **Open**
3. Select project folder
4. Wait for Gradle Sync

---

# 🔥 Firebase Setup

1. Create Firebase Project
2. Add Android App
3. Download `google-services.json`
4. Place file inside:

```text
app/google-services.json
```

---

# ▶️ Run Application

1. Connect Android Device or Emulator
2. Click ▶ Run button
3. Build APK if needed
