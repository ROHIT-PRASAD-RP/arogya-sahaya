# Arogya Sahaya

Arogya Sahaya is an Android healthcare assistance application designed to help users manage medicines, health records, reminders, and emergency support in a simple and user-friendly way.

---

## Features

### User Medical Profile

* Store personal health details
* Maintain patient information securely
* Easy access to emergency data

### Medicine Reminder

* Add medicines with schedules
* Receive reminder notifications
* Track upcoming doses

### Vital Tracker

* Record BP, Sugar, Pulse, and other vitals
* Maintain health history
* Visual health monitoring

### Emergency SOS

* Quick emergency support feature
* One-tap SOS functionality
* Fast access during emergencies

### Firebase Integration

* Cloud database support
* Secure data storage
* Real-time synchronization

### Modern UI

* Material Design interface
* Elderly-friendly layout
* Clean and responsive design

## Installation

### Clone Repository

```bash
git clone https://github.com/ROHIT-PRASAD-RP/arogya-sahaya.git
```

### Open in Android Studio

1. Open Android Studio
2. Click "Open"
3. Select the project folder
4. Sync Gradle

---

## Firebase Setup

1. Create a Firebase project
2. Add Android app
3. Download `google-services.json`
4. Place it inside the `app/` folder

---

## Build and Run

1. Connect Android device or emulator
2. Click Run ▶ in Android Studio
3. Build APK if needed
